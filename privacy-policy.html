[NAMA APLIKASI] ("Aplikasi") dikembangkan dan dikelola oleh [NAMA DEVELOPER] ("Kami"). Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pengguna sesuai dengan kebijakan Google Play dan peraturan yang berlaku.

1. Informasi yang Kami Kumpulkan

Untuk menggunakan fitur utama aplikasi, pengguna diwajibkan membuat akun dengan:

  • Username
  • Password

Kami tidak mengumpulkan data pribadi lain seperti kontak, lokasi, foto, pesan, atau informasi keuangan.

2. Tujuan Penggunaan Data

Informasi yang dikumpulkan digunakan untuk:

  • Proses autentikasi dan login
  • Manajemen akun pengguna
  • Menjalankan fungsi utama aplikasi
  • Menjaga keamanan sistem dan mencegah penyalahgunaan

3. Penyimpanan dan Keamanan Data

Data akun disimpan pada server yang dilindungi dengan langkah-langkah keamanan yang sesuai standar industri. Password tidak disimpan dalam bentuk teks biasa (plaintext), melainkan diproses menggunakan metode keamanan yang sesuai.

Seluruh transmisi data dilakukan melalui koneksi terenkripsi (HTTPS).

4. Izin Aplikasi

a. Izin Tampilkan di Atas Aplikasi Lain

Aplikasi dapat meminta izin untuk menampilkan tampilan di atas aplikasi lain. Izin ini digunakan secara terbatas untuk mendukung fungsi utama aplikasi. Aplikasi tidak memantau, merekam, atau mengambil data dari aktivitas pengguna di aplikasi lain.

b. Akses API Sistem Android

Aplikasi dapat mengakses API sistem Android yang diperlukan agar fitur berjalan dengan baik pada Android 12 dan versi di atasnya. Akses ini diberikan oleh pengguna dan hanya digunakan untuk menjalankan fungsi inti aplikasi. Aplikasi tidak mengakses atau mengumpulkan data dari aplikasi lain.

5. Pembagian Data

Kami tidak menjual, menyewakan, atau membagikan data pengguna kepada pihak ketiga. Data hanya dapat diungkapkan apabila diwajibkan oleh hukum.

6. Penyimpanan Data (Data Retention)

Data pengguna disimpan selama akun masih aktif. Pengguna dapat meminta penghapusan akun dan data terkait dengan menghubungi kami melalui email resmi.

7. Hak Pengguna

Pengguna berhak untuk:

  • Meminta akses terhadap data akun
  • Meminta perbaikan data
  • Meminta penghapusan akun

8. Perlindungan Anak

Aplikasi ini tidak ditujukan untuk anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan data dari anak-anak.

9. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan melalui halaman ini.
