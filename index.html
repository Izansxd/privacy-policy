Kebijakan Privasi

Siwo Prime

Tanggal Berlaku: 1 Maret 2025

Siwo Prime adalah aplikasi yang dikembangkan oleh Faza Alisky Ramadan. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi pengguna sesuai dengan kebijakan Google Play.

1. Informasi yang Kami Kumpulkan

Untuk menggunakan fitur aplikasi, pengguna diwajibkan membuat akun dengan:

  • Username
  • Password

Kami tidak mengumpulkan data pribadi lain seperti kontak, lokasi, foto, pesan, atau informasi keuangan.

2. Tujuan Penggunaan Data

  • Proses login dan autentikasi
  • Manajemen akun pengguna
  • Menjalankan fungsi utama aplikasi
  • Menjaga keamanan sistem dan mencegah penyalahgunaan

3. Penyimpanan dan Keamanan Data

Data akun disimpan di server yang aman. Password tidak disimpan dalam bentuk teks biasa (plaintext), melainkan diproses menggunakan metode keamanan yang sesuai standar industri.

Seluruh transmisi data dilakukan melalui koneksi terenkripsi (HTTPS).

4. Izin Aplikasi

a. Izin Tampilkan di Atas Aplikasi Lain

Aplikasi dapat meminta izin untuk menampilkan tampilan di atas aplikasi lain. Izin ini digunakan untuk mendukung fungsi utama aplikasi dan tidak digunakan untuk memantau atau merekam aktivitas pengguna di aplikasi lain.

b. Akses API Sistem Android

Aplikasi dapat mengakses API sistem Android yang diperlukan agar fitur berjalan dengan baik pada Android 12 dan versi di atasnya. Akses ini diberikan oleh pengguna dan hanya digunakan untuk menjalankan fungsi inti aplikasi.

5. Pembagian Data

Kami tidak menjual, menyewakan, atau membagikan data pengguna kepada pihak ketiga. Data hanya dapat diungkapkan apabila diwajibkan oleh hukum.

6. Penyimpanan Data

Data pengguna disimpan selama akun masih aktif. Pengguna dapat meminta penghapusan akun dan data dengan menghubungi kami melalui email di bawah ini.

7. Perlindungan Anak

Aplikasi ini tidak ditujukan untuk anak di bawah usia 13 tahun dan tidak secara sengaja mengumpulkan data dari anak-anak.
